# Feature Ideas Backlog

Last updated: `2026-03-04`

## Product-Aligned Priorities

1. Command palette search (`Cmd/Ctrl+K`) with instant node jump.
2. Goal mode with shortest-path guidance to a target concept.
3. Persistent exploration progression (visited/favorites/last position).
4. Frontier quests generated from underfilled nodes.
5. Node mastery states (`new`, `learning`, `mastered`) with filters.
6. Per-node notes and saved learning trails.
7. Prompt presets (`explain`, `practice`, `interview`, `project`).
8. Rich concept details panel (prerequisites, use-cases, pitfalls).
9. In-app data quality dashboard (cycles/fanout/coverage freshness).
10. One-click frontier-to-generation workflow from GUI.
11. Model bake-off mode for quality comparison across generators.
12. Shareable exploration links with selected node/camera state.

## Why These Fit Cyclop Eddy

- They reinforce the exploration + progression loop described in the manifest.
- They leverage current strengths (graph navigation, frontier tooling, quality reports).
- They prepare the product for account-based progression without requiring a full platform rewrite.
